# Overview
このドキュメントは、交通費精算アプリ（TERA: Transportation Expenses Reimbursement Application）の概要を説明します。

TERAは、EXP. サークルにおいて、メンバーが交通費を簡単に申請・管理し、精算までのプロセスを効率化することを目的としたウェブアプリケーションです。

## 技術スタック
- Next.js
- shadcn/ui
- Tanstack Query
- Tailwind CSS
- TypeScript
- Firebase (Authentication, Firestore)
- Vercel
- GitHub
